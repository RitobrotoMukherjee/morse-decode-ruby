# Morse Code Decoder

> This is a project with ruby that decodes morse code


## Built With

- ![](https://img.shields.io/badge/Ruby-critical)

## Getting Started

### To get this project up and running, you should do the following;
- If you wish to run the project locally, please do the following:

- Clone the repository to your local machine.
    - **`https://github.com/RitobrotoMukherjee/morse-decode-ruby.git`**
    - **`cd morse-decode-ruby`**

## Prerequisite

> ### Ruby needs to be installed in system
